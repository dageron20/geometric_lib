# Math formulas

## Area

- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter

- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a

# Общее описание решения

### В этом проекте представлены такие формулы как:

- формулы нахождения площади окружности, прямоугольника, квадрата и треугольника
- формулы нахождения периметра окружности, прямоугольника, квадрата и треугольника

# Описание работы файла circle.py

## _Описание работы функции area(r):_

- Возвращает площадь окружности радиуса `r`.

### Параметры, передаваемые в функцию:

- `r (float)` - радиус окружности

### Возвращаемое значение:

- Площадь окружности `(S = π * r * r)`

### Пример вызова функции

- `area(5)` = 78.53981633974483

## _Описание работы функции perimeter(r):_

- Возвращает длину окружности радиуса `r`.

### Параметры, передаваемые в функцию:

- `r (float)` - радиус окружности

### Возвращаемое значение:

- Длина окружности `(P = 2 * π * r)`

### Пример вызова функции

- `perimeter(5)` = 31.41592

# Описание работы файла rectangle.py

## _Описание работы функции area(a, b):_

- Возвращает площадь прямоугольника со сторонами `a` и `b`.

### Параметры, передаваемые в функцию:

- `a (float)` - сторона прямоугольника
- `b (float)` - сторона прямоугольника

### Возвращаемое значение:

- Площадь прямоугольника `(S = a * b)`

### Пример вызова функции

- `area(2, 3)` = 6

## _Описание работы функции perimeter(a, b):_

- Возвращает периметр прямоугольника со сторонами `a` и `b`.

### Параметры, передаваемые в функцию:

- `a (float)` - сторона прямоугольника
- `b (float)` - сторона прямоугольника

### Возвращаемое значение:

- Периметр прямоугольника `(P = 2 * (a + b))`

### Пример вызова функции

- `area(2, 3)` = 10

# Описание работы файла triangle.py

## _Описание работы функции area(a, h):_

- Возвращает площадь треугольника со стороной `a` и высотой, проведённой к этой стороне `h`.

### Параметры, передаваемые в функцию:

- `a (float)` - сторона треугольника
- `h (float)` - высота треугольника

### Возвращаемое значение:

- Площадь треугольника `(S = a * h / 2)`

### Пример вызова функции

- `area(5, 6)` = 15

## _Описание работы функции perimeter(a, b, c):_

- Возвращает периметр треугольника со сторонами `a`, `b` и `c` .

### Параметры, передаваемые в функцию:

- `a (float)` - сторона треугольника
- `b (float)` - сторона треугольника
- `c (float)` - сторона треугольника

### Возвращаемое значение:

- Периметр треугольника `(P = a + b + c)`

### Пример вызова функции

- `area(5, 6, 7)` = 18
