- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a

# Unit тесты 
## Цели и задачи тестирования: Круг
В процессе тестирования необходимо проверить работу кода. Работает ли он корректно проверим с помощью полученния одинаковых результатов при подсчете вручную и при подсчете с помощью **Unit тестов**.


## Описание тестируемого продукта 
Для вычисления площади и периметра круга известны две формулы: 

```python

def area(r):
return math.pi * r * r

def perimeter(r):
return 2 * math.pi * r

```

В обоих случаях для вычесления используются радиус - **r** и число **pi** - math.pi.

## Область тестирования 
В процессе тестирования будет проверено работают ли корректно функции кода для вычисления площади и периметра круга. Результат **Unit теста** должен совпадать с результатом полученным вручную.

## Стратегии тестирования 
Для начала определяем набор входных данных для вычисления площади: 

    res = area(math.pi, 2)

И периметра:

    res = area(2, math.pi, 2)

Зная алгоритм работы кода вычисляем вручную площадь и периметр круга. Записываем входные данные и результат подсчета в код. 

     res = area(math.pi, 2)
     self.assertEqual(res, 2 math.pi)

     res = area(2, math.pi, 2)
     self.assertEqual(res, 4 math.pi)

Далее запускаем работу кода и сравниваем полученный ответ. Если ответы совпадают, то код работает корректно. Если нет, то записываем ошибку в протокол.

## Критерии приемки
Для успешного завершения тестирования и приемки продукта ответ из **Unit тестов** должны совпадать с результатами полученными вручную. В таком случае продукт работает корректно.
