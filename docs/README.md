# Задача проекта
Проект реализует формулы площади и периметра для некоторых многоугольников. Для каждой фигуры выделена своя функция.

### Фигуры:
- rectangle (прямоугольника)
- square (квадрат)
- triangle (треугольник)
- circle (окружность)

# Описание решения
__Ввод данных:__ числа с палавающей точкой _float_ (пользователю можно использовать целочисленные числа)

__Вывод данных:__ того же формата что и ввод

В проекте был использована стандартная библиотека __math__ для использования числа __pi__

# Описание функций 
## Файл rectangle.py
В файле приведены функции для подсчета площади и периметра прямоугольника
### Содержимое файла
```
def area(a, b):
    '''
    параметры:
    a (float, int): первая сторона (число)
    b (float, int): вторая сторона (число)

    возвращаемое значение:
    площадь прямоугольника по формуле через две стороны

    при вводе отрицательных чисел выводит ошибку
    '''
    if isinstance (a, str) or isinstance (b, str):
        return 'Вы ввели строку'
    if a < 0 or b < 0:
        return 'Сторона прямоугольника не может быть отрицательной'
    return a * b

def perimeter(a, b):
    '''
    параметры:
    a (float, int): первая сторона (число)
    b (float, int): вторая стороны (число)

    возвращаемое значение:
    периметр прямоугольника по формуле через две стороны

    при вводе отрицательных чисел выводит ошибку
    '''
    if isinstance (a, str) or isinstance (b, str):
        return 'Вы ввели строку'
    if a < 0 or b < 0:
        return 'Сторона прямоугольника не может быть отрицательной'
    return (a + b) * 2
```

- функция __area__ отвечает за нахождение площади
- функция __perimeter__ отвечает за нахождение периметра

### Пример вызова
**print(area(5, 3))** 
- 15

**print(perimeter(5, 3))**
- 16

**print(area(2.5, 6.2))** 
- 15.5

**print(perimeter(2.5, 6.2))**
- 17.4

## Файл triangle.py
В файле приведены функции для подсчета площади и периметра треугольника
### Содержимое файла
```
def area(a, h):
    '''
    параметры:
    a (float, int): сторона треугольнаа (число)
    h (float, int): высота к стороне a (число)
    возвращаемое значение:
    площадь треугольника по формуле через сторону и прилежащую к ней высоту

    при вводе отрицательных чисел выводит ошибку
    '''
    if isinstance (a, str) or isinstance (h, str):
        return 'Вы ввели строку'
    if a < 0 or h < 0:
        return 'Сторона или высота треугольника не может быть отрицательным числом'
    return a * h / 2

def perimeter(a, b, c):
    '''
    параметры:
    a (int): сторона треугольника (число)
    b (int): сторона треугольника (число)
    c (int): сторона треугольника (число)
    возвращаемое значение:
    сумма сторон (периметр треугольника) по формуле трех сторон

    при вводе отрицательных или нулевых чисел выводит ошибку
    '''
    if isinstance (a, str) or isinstance (b, str) or isinstance (c, str):
        return 'Вы ввели строку'
    if a <= 0 or b <= 0 or c <= 0:
        return 'Сторона треугольника не может быть отрицательным числом или нулем'
    return a + b + c
```
- функция __area__ отвечает за нахождение площади
- функция __perimeter__ отвечает за нахождение периметра
### Пример вызова
**print(area(5, 4))** 
- 10

**print(perimeter(5, 4, 3))**
- 12

**print(area(5.3, 4.2))** 
- 11.13

**print(perimeter(5.1, 4.1, 3.1))**
- 12.3

## Файл circle.py
В файле приведены функции для подсчета площади и периметра круга

в функциях и спользовалась стандартная библиотека
### Содержимое файла
```
import math

def area(r):
    '''
    с помощью библиотеки math задается число pi
    параметры:
    r (float, int): радиус окружности (число)
    возвращаемое значение:
    площадь круга с заданным радиусом по формуле

    при вводе отрицательного числа выдает ошибку
    '''
    if isinstance (r, str):
        return 'Вы ввели строку'
    if r < 0:
        return 'Радиус не может быть отрицательным числом'
    return math.pi * r * r


def perimeter(r):
    '''
    с помощью библиотеки math задается число pi
    параметры:
    r (int): радиус окружности (число)
    возвращаемое значение:
    периметр окружности с заданным радиусом по формуле

    при вводе отрицательного числа выдает ошибку
    '''
    if isinstance (r, str):
        return 'Вы ввели строку'
    if r < 0:
        return 'Радиус не может быть отрицательным числом'
    return 2 * math.pi * r

```
- функция __area__ отвечает за нахождение площади
- функция __perimeter__ отвечает за нахождение периметра
### Пример вызова
**print(area(5))** 
- 78.5

**print(perimeter(5))**
- 31.4
