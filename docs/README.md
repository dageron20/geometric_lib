# Math formulas
## Area
- Circle: S = πR²
###
import math
def area(r):
  return r * math.pi * r
print(area(5))
Output: 78.53981633974483 
- Rectangle: S = ab
###
def area(a, b)
  return a * b
print(area(4, 5))
Output: 20
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
